## 2024-04-21 - Highlight Keyboard Shortcuts in UI Help Text
**Learning:** Terminal UI users often miss keyboard shortcuts hidden within plain gray helper text. Drawing explicit visual attention to interaction keys makes the interface significantly more discoverable.
**Action:** When adding or updating keyboard shortcuts in helper text, wrap the specific keys (e.g., 'Enter', 'Tab', 'Up/Down') in a prominent visual component like `<Text color="white" bold>` against the standard `gray` background text.
