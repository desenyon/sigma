## 2024-04-19 - Keyboard Shortcut Highlighting
**Learning:** Users struggle to discover keyboard navigation shortcuts in CLI interfaces when they are styled uniformly with helper text. Wrapping inline keyboard shortcuts in contrasting, bold styles (e.g., `<Text color="white" bold>`) against muted text (e.g., `<Text color="gray">`) significantly improves keyboard discoverability and visual hierarchy without cluttering the UI.
**Action:** Apply this styling pattern to all future inline keyboard shortcut references in Ink terminal UIs.
