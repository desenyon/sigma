## 2024-04-22 - [Add keyboard shortcut hints to Ink UI]
**Learning:** Highlighting keyboard shortcuts using a bolder and different color compared to surrounding gray helper text significantly improves keyboard discoverability in a CLI environment.
**Action:** Consistently wrap inline keyboard shortcut references (e.g., 'Tab', 'Enter', 'Up/Down') in a `<Text color="white" bold>` component to draw visual attention without being overpowering.
