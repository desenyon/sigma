## 2024-03-24 - Highlighted Keyboard Shortcuts
**Learning:** In the Ephemeral Ink terminal UI, plain gray text for keyboard shortcuts is hard to parse quickly. A reusable UX pattern for this design system is to isolate keycap names using `<Text color="white" bold>` inline with `<Text color="gray">` descriptions.
**Action:** Always highlight terminal UI keyboard shortcuts (e.g. Tab, ↑/↓, d) in a contrasting bold color to improve scannability in dense layouts.
